# Next.js Development Skill

## Description
This skill provides expertise in developing applications using Next.js, a popular React framework for building production-ready web applications.

## Capabilities
- Create and configure Next.js applications
- Implement page routing and API routes
- Handle server-side rendering (SSR) and static site generation (SSG)
- Manage data fetching strategies
- Configure Next.js build and deployment settings
- Implement authentication and authorization
- Optimize performance and SEO
- Handle forms and user input
- Integrate with databases and external APIs
- Apply styling with CSS modules, Tailwind, or other solutions

## Best Practices
- Follow Next.js file-based routing conventions
- Use getStaticProps/getStaticPaths for static generation
- Use getServerSideProps for server-side rendering
- Implement proper error handling
- Optimize images using Next.js Image component
- Use dynamic imports for code splitting
- Follow accessibility guidelines
- Implement proper meta tags for SEO

## Common Commands
- `npx create-next-app@latest` - Create a new Next.js app
- `npm run dev` - Start development server
- `npm run build` - Build the application
- `npm start` - Start production server

## File Structure
- `pages/` - Contains page components
- `public/` - Static assets
- `styles/` - Global styles
- `components/` - Reusable React components
- `api/` - API routes (in pages router)
- `app/` - App directory (in app router)
