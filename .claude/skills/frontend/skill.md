# Frontend Development Skill

## Description
This skill provides expertise in modern frontend web development, covering HTML, CSS, JavaScript, and popular frameworks/libraries for building responsive and interactive user interfaces.

## Capabilities
- Create semantic and accessible HTML markup
- Style interfaces with CSS, preprocessors, and frameworks
- Implement interactive behavior with vanilla JavaScript
- Work with popular frameworks (React, Vue, Angular)
- Build responsive designs for all screen sizes
- Implement modern CSS methodologies (BEM, SMACSS)
- Handle state management in frontend applications
- Create reusable and modular components
- Optimize frontend performance
- Implement proper error handling and validation
- Ensure cross-browser compatibility
- Apply accessibility best practices (WCAG)

## Best Practices
- Write semantic HTML for better accessibility
- Use CSS variables for consistent theming
- Implement mobile-first responsive design
- Follow component-based architecture
- Use proper event handling and delegation
- Implement lazy loading for performance
- Minimize bundle sizes
- Use modern JavaScript features appropriately
- Follow naming conventions consistently
- Write maintainable and scalable code

## Technologies & Tools
- HTML5, CSS3, JavaScript (ES6+)
- CSS Frameworks: Bootstrap, Tailwind, Bulma
- Preprocessors: Sass, Less, Stylus
- Frontend Frameworks: React, Vue, Angular
- Build Tools: Webpack, Vite, Parcel
- Package Managers: npm, yarn
- Testing: Jest, Cypress, Enzyme
- Linting: ESLint, Stylelint

## Common Patterns
- Component lifecycle management
- Conditional rendering
- Form handling and validation
- API integration and data fetching
- Routing and navigation
- State management patterns
- Animation and transitions
- Progressive Web Apps (PWA)
- Single Page Applications (SPA)
